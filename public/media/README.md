# Patriot K9 Media Library

This folder is reserved for authentic Patriot K9 website assets. Organize new media by subject so future page updates can pull from a clean, consistent source library.

## Folder Guide

- `homepage`: strongest homepage-only assets
- `puppies`: puppy portraits, litter photos, puppy development
- `breeding`: sire, dam, litter, pedigree-related visuals
- `training/obedience`: sit, down, place, stay, recall, heel
- `training/public-training`: stores, offices, parks, sidewalks, public manners
- `training/behavior-modification`: reactivity, neutrality, aggression rehabilitation
- `training/puppy-foundation`: early obedience, engagement, socialization
- `training/confidence`: obstacles, surfaces, environmental confidence
- `facility`: kennel room, property, training grounds
- `lifestyle`: trainer with dogs, daily life, travel
- `community`: veteran and community-related photos
- `team`: trainer, breeder, staff, partners
- `testimonials`: client result photos and graduation photos
- `video-posters`: poster images used before videos play

## Naming Rules

- use lowercase filenames
- use hyphens instead of spaces
- use descriptive names
- do not include dates unless needed
- do not use random camera names like `IMG_1234`
